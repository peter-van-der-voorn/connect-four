# connect-four
My take on the 1974 Milton Bradley classic Connect Four game

# Setup


* Clone down the repo

  ```sh
  git clone https://github.com/peter-van-der-voorn/connect-four.git
  ```

* Install dependencies

  ```sh
  npm i
  ```

* Spin up the local server

  ```sh
  npm run dev
  ```

* Check it out in the browser

  http://localhost:3000/


* Enjoy!
